# AI Mental Stress Detector - Streamlit Version

A Streamlit-ready AI-style mental stress detector web app with:

- Login and registration
- Stress test form
- Questionnaire-based stress scoring
- AI-style result generation
- Report download
- Stress history tracking
- Wellness tips
- Mental wellness chatbot
- Profile and settings pages
- SQLite database persistence

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Community Cloud

1. Upload this folder to GitHub.
2. Go to Streamlit Community Cloud.
3. Create a new app.
4. Select your repository and branch.
5. Set main file path to:

```text
app.py
```

6. Click Deploy.

## Important

This version is converted from the original Flask project. The original Flask file is kept as `flask_app_original.py` for reference, but Streamlit will run `app.py`.
